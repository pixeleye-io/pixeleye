# Pixeleye CLI

Pixeleye CLI is a command line interface for Pixeleye. It makes uploading snapshots to Pixeleye a breeze.

## Installation

- npm

  ```sh
  npm install -g @pixeleye/cli
  ```

- yarn

  ```sh
  yarn global add @pixeleye/cli
  ```

- pnpm
  ```sh
    yarn global add @pixeleye/cli
  ```
